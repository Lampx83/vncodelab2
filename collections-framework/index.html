Tạo một lớp Student

public class Student {
   String StudentName;
   String StudentID;
   String description;
   int Studentage;

   public Student() {
   }

   public Student(String pName, String pID, String descr, int age) {
       StudentName = pName;
       StudentID = pID;
       description = descr;
       Studentage = age;
   }

   public String toString() {
       return "Student Name: " + StudentName + "\tStudent ID: " + StudentID
               + "\tDescription: " + description + "\tage: " + Studentage;
   }
}

Tạo lớp PeopList

import java.util.ArrayList;

public class PeopleList<T> {
   ArrayList<T> listPeople;

   public PeopleList() {
       listPeople = new ArrayList<T>();
   }

   public void add(T obj) {
       listPeople.add(obj);
   }

   public void display() {
       for (T objPeople : listPeople) {
           System.out.println(objPeople.toString());
       }
   }

   public int getSize() {
       return listPeople.size();
   }

   public boolean checkEmpty() {
       return (listPeople.size() == 0);
   }

   public T grab() {
       if (!checkEmpty()) {
           return listPeople.remove(0);
       } else
           return null;
   }
}

Viết hàm main sử dụng 2 class trên

public static void main(String[] args) {
   PeopleList<Student> objStudent = new PeopleList<Student>();
   objStudent.add(new Student("Pham Xuan Minh", "123", "Class1", 22));

   System.out.println("Size of the list: " + objStudent.getSize());

   System.out.println("************");

HIển thị các phần tử:

System.out.print("Linked list data: ");

   System.out.print(iterator.next() + " ");
}

Kết quả:

Iterator có thể làm được điều mà fori và foreach không thể làm được. Ví dụ như xóa một phần tử trong quá trình lặp.

Ví dụ duyệt, kiểm tra, xóa:

Iterator<Integer> iterator = list.iterator();
while (iterator.hasNext()) {
   Integer num = iterator.next();
   if (num < 3)
       iterator.remove();
}
